\title{The Median-Of-Medians Selection Algorithm}
\author{Manuel Eberl}
\maketitle

\begin{abstract}
This entry provides an executable functional implementation of the Median-of-Medians algorithm~\cite{clrs2009} for selecting the $k$-th smallest element of an unsorted list deterministically in linear time. The size bounds for the recursive call that lead to the linear upper bound on the run-time of the algorithm are also proven.
\end{abstract}
