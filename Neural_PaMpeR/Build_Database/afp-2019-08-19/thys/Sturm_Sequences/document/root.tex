\title{A Formalisation of Sturm's Theorem}
\author{Manuel Eberl}
\maketitle

\begin{abstract}
\emph{Sturm sequences} are a method for computing the number of real roots
of a real polynomial inside a given interval efficiently. In this project,
this fact and a number of me\-thods to construct Sturm sequences efficiently
have been formalised with the interactive theorem prover Isabelle\slash HOL.
Buil\-ding upon this, an Isabelle\slash HOL proof method was then
implemented to prove statements about the number of roots of a real
polynomial and related properties.
\end{abstract}
